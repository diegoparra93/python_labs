# Python Labs

## Task 1: Greeting and Age
**File:** `src/01_greeting.py`  
**Description:** This program asks for the user's name and age, then greets them and says how old they will be next year.

## Task 2: Sum and Average
**File:** `src/02_sum_avg.py`  
**Description:** This program takes two decimal numbers, calculates their sum and average, and displays the results with two decimal places.  
![Task 2 Screenshot](images/02_sum_avg.png)

## Task 3: Discount and VAT Calculation
**File:** `src/03_discount_vat.py`  
**Description:** This program calculates the final price of a product after applying a discount and adding value-added tax (VAT).  
![Task 3 Screenshot](images/03_discount_vat.png)

## Task 4: Minutes to Hours and Minutes
**File:** `src/04_minutes_to_hhmm.py`  
**Description:** Converts total minutes into hours:minutes format (HH:MM).

## Task 5: Initials and Name Length
**File:** `src/05_initials_and_len.py`
